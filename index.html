HTML (Hyper-Text Markup Language): important tags

You don't have to remember these tags, because you can always search for the rigth tags using searchmachines like Google, but I think it's important to know at least some tags, like:

anchor: a
break: br
paragraph: p
insert a link

Some other important things to remember:


Element: An HTML element refers to everything within a set of opening and closing tags.
Attribute: This is a property of an HTML element. For example, to set the href attribute of an anchor tag to an URL
Inline vs. block: A block makes an invisable box.
